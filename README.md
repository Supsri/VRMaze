# VRMaze
A virtual reality maze with customisable walls.

In order to change the lining of the maze walls, open up the project in Unity, replace the file https://github.com/Supsri/VRMaze/blob/master/MazeScene/Assets/Cope!%20Free%20Skybox%20Pack/materials/boo.jpg with an image of your choice.

You will need: Android SDK, GVR(google VR) added as a package/plugin to unity.

You an add an mp3 asset as well, for your game music.
